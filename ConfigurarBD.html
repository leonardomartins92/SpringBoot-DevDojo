Configurar Banco de dados:

1. Adicionar dependências

org.springframework.boot spring-boot-starter-data-jpa mysql mysql-connector-java

2. Adicionar configurar resources/application.yml

server: error: include-stacktrace: on_param spring: datasource: url: jdbc:mysql://localhost:3306/anime?useSSL=false&createDatabaseIfNotExist=true&allowPublicKeyRetrieval=true username: root password: root jpa: hibernate: ddl-auto: update

3. Configurar as classes que serão Entidades:

@Entity public class Anime{ @Id @GeneratedValue(strategy = GenerationType.IDENTITY) }
